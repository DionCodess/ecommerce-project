\## App description



CS491 Market is a TypeScript React marketplace application connected to UN Sustainable Development Goal 12: Responsible Consumption and Production. The app demonstrates a sustainable shopping workflow where users can browse products, search and filter the catalog, add items to a cart, complete a dummy checkout, and manage products and orders through an admin dashboard.



\## Main features



\* Product catalog

\* Search and category filtering

\* Product detail pages

\* Shopping cart

\* Dummy checkout

\* Admin dashboard

\* Product management

\* Order management

\* Firebase configuration

\* Firestore rules and indexes

\* Firebase Auth support

\* Vitest unit tests
